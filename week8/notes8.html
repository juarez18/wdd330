Canvas, SVG and Drag and Drop

Sorry! your browser doesn't support Canvas

Canvas and SVG are very similar but each can be useful depending on the situation. Canvas allows for pixel manipulation which has pretty unique feature but one downside is immediate mode. If you want to add to what you already drawn on canvas you must redraw the new shape from scratch but you can still save images. Although Canvas is much faster so it is a great option when making something that requires animation. SVG on the other had is accessible via dom and is preserved in the document although it is much harder to manipulate pixels and such.
